The code was created using the PyCharm IDE on an Anaconda Distribution
Before runnung the script, make sure that python 3.7 is installed. 
Then install the following libraries
pandas 0.24.2
numpy 1.16.2
matplotlib 3.03
scikit-learn 0.20.3

The code folder has three python scripts. Namely, Manhattan Detector, visual and driver.
Along with the code, an excel file of the dataset is attached.

Run the driver.py file making sure that the dataset is in the same directory as the code

The code prompts for 2 inputs, threshold and sample size.

The driver.py display results in the console
